# Fit-Forge

Fit-Forge is a clean and efficient website that allows users to learn and understand different exercises. 

# Features
- Search for exercises either using muscle groups or exercise names
- Includes related tags for each workout to easily group exercises
- YouTube videos related to each exercise to provide visualization and demonstration
- Similar exercises section to work on the same muscle groups

# Installation
1. Clone this repository
2. Install dependancies and run the server
```sh
npm install
npm run
```
